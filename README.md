# free_swimming_tail_tracking
Tail tracking software designed for analyzing larval zebrafish behaviour. Written in [Python](https://www.python.org/). Contains utility functions, user interface, and data analysis/plotting tools for eye tracking and tail tracking.

## Dependencies
Requires [Python 3.6.1](https://www.python.org/downloads/release/python-361/) and [OpenCV](https://opencv.org/) installed.

## Usage
UT contains functions for tracking. GUI generates user interface. Examples of plotting and command line scripts in PLOT and CL.
